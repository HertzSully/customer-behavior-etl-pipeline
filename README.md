# Customer Behavior ETL-pipeline
An end-to-end data engineering and analytics project transforming raw e-commerce data into actionable insights using Python (Pandas/SQLAlchemy), PostgreSQL, and Power BI

# Features
ETL Pipeline: Automated data cleaning and transformation using Python and Pandas.

Database Ingestion: Seamless data transfer from CSV to a PostgreSQL warehouse.

Advanced SQL: Complex analytics using CTEs and Window Functions to rank product performance.

BI Dashboard: Interactive Power BI report visualizing revenue by age group and customer segments.

# Tech Stack
Language: Python (Pandas, SQLAlchemy).

Database: PostgreSQL.

Visualization: Power BI.

# Project Structure
Data Cleaning: Handled missing values and normalized column names in Python.

Feature Engineering: Created age groups and mapped purchase frequencies to numerical values.

SQL Analysis: Identified the Top 3 best-selling items per category using ROW_NUMBER().

Reporting: Developed a financial summary dashboard to track revenue distribution.

# How to Run
Ensure a PostgreSQL instance is running on localhost:5432.

Run the Jupyter Notebook to process and upload the data.

Open the Power BI file to view the analysis.
